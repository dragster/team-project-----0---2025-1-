# Описание проекта
Простой веб-калькулятор реализует базовые арифметические операции (+, -, ×, ÷) с сохранением истории вычислений в localStorage браузера. 
Проект демонстрирует навыки frontend-разработки, Git workflow и Python-скриптинга для автоматизации.

# Функционал:
Базовые операции с приоритетом (PEMDAS)
История до 50 вычислений с очисткой
Responsive дизайн (мобильные/десктоп)
Обработка ошибок (деление на ноль)
Клавиатурный ввод (Enter, цифры, операции)

# Горячие клавиши:
text
0-9    Цифры
+ - * / Операции
Enter  = (вычислить)
Escape Очистить
h/H    История

# В браузере
1. Откройте index.html
2. Нажимайте цифры и операции
3. Enter или = для результата
4. Кнопка "История" — просмотр/очистка
