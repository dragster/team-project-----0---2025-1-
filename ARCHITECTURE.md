# Документация функций/методов: Простой калькулятор
# Описание проекта
#Назначение: Веб-приложение калькулятор с поддержкой базовых операций и историей вычислений в localStorage браузера. Демонстрирует навыки frontend-разработки, Git workflow и автоматизации.

# Основной функционал:
Арифметические операции: +, -, *, /
История вычислений (до 50 записей)
Responsive интерфейс
Обработка ошибок (деление на ноль)
Клавиатурный ввод
